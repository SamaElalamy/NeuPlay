# 🧠 NeuPlay

An interactive, gamified desktop application designed to support and empower children dealing with learning difficulties, specifically **Dysgraphia** and **Dyscalculia**. 

NeuPlay provides a safe and  engaging environment where children can be assessed, learn, and grow through gamification, while building their confidence step-by-step.

---

## 🎯 The Problem It Solves
Many children struggle with traditional educational methods due to SLD (specific learning difficulties):
- **Dysgraphia:** It causes significant difficulty with the physical act of writing , spelling and organizing thoughts on paper.
- **Dyscalculia:** Difficulty understanding numbers, math concepts, visualizing quantities, comparing them and performing calculations.

**NeuPlay** Bridges Learning Gaps in Dysgraphia and Dyscalculia through Implementing Gamification Mechanics

---

## ✨ Key Features
- **🧠 Smart Diagnosis Test:** An assessment tool that evaluates the child and determines their specific difficulty (Dysgraphia, Dyscalculia, or Both).
- **🎮 Educational Games:**  **Spell Quest** Interactive levels including Flashcards, Fill-in-the-blanks, and Drag-and-Drop exercises, **Numbers Land** Interactive levels including Drag-and-Drop, Visualization and Sequence-based levels .
- **🏆 Reward System:** A  tracking system that awards points, calculates streaks, and recognizes resilience through badges to boost self confidence.
- **🗄️ Custom Local Database:** A functional File I/O system (Text-based database) that registers users, handles authentication, and reads level data dynamically.
---

## 🛠️ Technologies Used
- **Language:** C#
- **Framework:** .NET Windows Forms (WinForms)
- **Data Management:** System.IO (Custom File Handling)

---

## 🏗️ Software Engineering & OOP Concepts
NeuPlay was built with a strong focus on clean architecture and Object-Oriented Programming principles:
- **Encapsulation:** Protecting game states and user data using private fields and secure access modifiers.
- **Abstraction:** Hiding complex file reading and data parsing logic behind simple `DataManager` methods.
- **Polymorphism:** Utilizing method overloading in class constructors to handle different UI states dynamically.
